What's changing — full list

  • Structure fix: Filter bar and table are proper siblings inside the card (not nested in a flex-wrap)
  • Row actions fade: Edit/delete buttons are dim (opacity 20%) until you hover the row — less visual noise
  • Subtle stripe: Every other row gets a very light bg tint for scannability
  • Shorter date on mobile: "12. mars" instead of "12. mars 2026" — year visible in the period filter above
  • Category pill in description on mobile: Shows inline below the name instead of a separate column
  • Hidden columns on mobile: Category, Account, Actions columns hidden at <md — tap row for sheet
  • Both tables: Expenses and Income get the same treatment
